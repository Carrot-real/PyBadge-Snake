# PyBadge-Snake
This is the game snake coded for a PyBadge LC. I coded this in CircuitPython version 10.0.3. The version of CircuitPython used is a slightly modified version for slight efficiency increase and fixed stage and ugame library's.

# Features
This snake game features lots of options. There are 9 backgrounds, 11 customizable snake colors, 10 speed options, shadows, and a face which are all customizable through the built in settings menu which you can access by pressing the select button.
